**DocuNova 배포를 위한 플랫폼 비교 가이드**

## 🎯 한눈에 비교

| 항목 | AWS | 네이버 클라우드 | 승자 |
|------|-----|---------------|------|
| **난이도** | ⭐⭐⭐ 중간 | ⭐⭐ 쉬움 | 🏆 NCP |
| **한글 지원** | 부분 지원 | 완벽 지원 | 🏆 NCP |
| **기술 지원** | 영어/채팅 | 한국어/전화 | 🏆 NCP |
| **설정 복잡도** | 복잡 (IAM, VPC 등) | 단순 | 🏆 NCP |
| **속도 (한국)** | 빠름 | 매우 빠름 | 🏆 NCP |
| **가격** | ⭐⭐⭐ 비쌈 | ⭐⭐ 저렴 | 🏆 NCP |

| **확장성** | 무한 | 높음 | 🏆 AWS |
| **문서 품질** | 방대 (영어) | 적당 (한글) | 비김 |

## 🏆 결론: 어느 것을 선택해야 할까?

### ✅ 네이버 클라우드를 선택하세요:
- 👍 **클라우드 초보자**
- 👍 **한국어 지원이 필요한 경우**
- 👍 **빠른 전화/이메일 지원이 필요한 경우**
- 👍 **국내 사용자만 대상** (한국 서비스)
- 👍 **비용을 절약하고 싶은 경우**
- 👍 **간단하고 빠른 설정을 원하는 경우**

### ✅ AWS를 선택하세요:
- 👍 **클라우드 경험이 있는 경우**
- 👍 **글로벌 서비스를 계획 중**
- 👍 **다양한 AWS 서비스 활용 예정** (S3, Lambda, RDS 등)
- 👍 **장기적인 확장성이 중요한 경우**
- 👍 **1년 프리티어를 활용하고 싶은 경우**

| **고정 IP** | 무료 (사용 중) | ₩3,000 |
| **스토리지 (50GB)** | 포함 | 포함 |

| **총 월 비용** | **₩95,000** | **₩68,000** |
| **12개월 비용** | ₩1,140,000 | ₩816,000 |
| **차이** | - | **₩324,000 절약** |

### 프리티어 활용 시

| 항목 | AWS (12개월) | 네이버 (신규) |
|------|-------------|-------------|
| **무료 혜택** | 750시간/월 (t3.micro) | 크레딧 100,000원 |
| **제약사항** | t3.micro는 메모리 1GB (부족!) | 모든 서버 타입 가능 |
| **실질 무료 기간** | 12개월 (단, 작은 서버만) | 약 1.5개월 |

**주의**: DocuNova는 Ollama LLM 실행을 위해 **최소 8GB RAM** 필요!
→ AWS 프리티어 t3.micro (1GB RAM)는 **사용 불가**
→ 실제로는 t3.large 이상 필요 → **프리티어 혜택 없음**

## 🎓 난이도 비교

### 네이버 클라우드 (쉬움 ⭐⭐)

**장점:**
- ✅ 콘솔이 직관적이고 한글
- ✅ 설정 항목이 적음
- ✅ VPC, ACG 설정이 간단
- ✅ 전화/이메일 한국어 지원
- ✅ 가이드가 한글로 잘 되어있음

**단점:**
- ⚠️ AWS보다 기능이 제한적
- ⚠️ 영문 자료가 적음
- ⚠️ 글로벌 리전 없음

**배포 시간**: 약 45분

---

### AWS (중간 ⭐⭐⭐)

**장점:**
- ✅ 매우 방대한 기능
- ✅ 무한한 확장성
- ✅ 많은 레퍼런스 및 커뮤니티
- ✅ 글로벌 리전 (전 세계)

**단점:**
- ⚠️ IAM (권한 관리) 복잡
- ⚠️ VPC, 서브넷, 보안그룹 설정 복잡
- ⚠️ 용어가 어려움 (EC2, EBS, ELB 등)
- ⚠️ 한글 지원 부족
- ⚠️ 기술 지원 유료 (월 $29부터)

**배포 시간**: 약 60-90분 (설정이 복잡함)

## 🔧 설정 복잡도 비교

### 1. 서버 생성

| 단계 | AWS | 네이버 클라우드 |
|------|-----|---------------|
| **VPC 생성** | 복잡 (CIDR, 라우팅 테이블 등) | 간단 (IP 범위만) |
| **서브넷 설정** | 복잡 (Public/Private 구분) | 간단 |
| **보안 그룹** | Security Group (복잡) | ACG (간단) |
| **키 페어** | 동일 | 동일 |
| **인스턴스 생성** | 옵션 많음 (헷갈림) | 옵션 적음 (명확) |

### 2. 네트워크 설정

| 항목 | AWS | 네이버 클라우드 |
|------|-----|---------------|
| **고정 IP** | Elastic IP (복잡) | Public IP (간단) |
| **로드밸런서** | ELB (복잡, 여러 종류) | Load Balancer (간단) |
| **방화벽** | Security Group + NACL | ACG만 |

### 3. 스토리지

| 항목 | AWS | 네이버 클라우드 |
|------|-----|---------------|
| **볼륨** | EBS (복잡) | Block Storage (간단) |
| **스냅샷** | EBS Snapshot | Snapshot |
| **파일 스토리지** | S3 (버킷, 정책 등 복잡) | Object Storage (간단) |

## 🌍 지역(Region) 비교

### AWS
- **서울 리전**: ap-northeast-2
- **글로벌**: 전 세계 30개 이상 리전
- **가용영역**: 서울 4개 (a, b, c, d)

### 네이버 클라우드
- **한국 리전**: KR-1, KR-2 (서울, 부산)
- **글로벌**: 싱가포르, 홍콩, 미국 등 (제한적)
- **가용영역**: 2개

**결론**:
- 한국만 서비스 → NCP 충분
- 글로벌 서비스 → AWS 필수

## 🎁 프리티어 상세 비교

### AWS 프리티어 (12개월)

**무료 제공:**
- EC2: t3.micro (1 vCPU, 1GB RAM) 750시간/월
- EBS: 30GB
- S3: 5GB
- RDS: db.t3.micro 750시간/월
- Lambda: 100만 요청/월

**문제점:**
- ❌ DocuNova는 **최소 8GB RAM** 필요
- ❌ t3.micro (1GB)로는 **Ollama 실행 불가**
- ❌ t3.large (8GB) 사용 시 **프리티어 혜택 없음**
- 💰 **실제 비용**: 월 약 $60-80 (₩80,000-100,000)

### 네이버 클라우드 프리티어

**무료 제공:**
- 신규 가입 크레딧: ₩100,000
- 모든 서버 타입에 사용 가능
- 유효기간: 3개월

**장점:**
- ✅ 8GB RAM 서버에도 사용 가능
- ✅ 약 1.5개월 무료 운영
- ✅ 제약 없음

## 💡 추천 시나리오

### 🎯 시나리오 1: 개인 프로젝트 / 스타트업 MVP

**상황**:
- 클라우드 초보
- 빠르게 배포하고 테스트하고 싶음
- 예산이 제한적
- 한국 사용자만 대상

**추천**: **네이버 클라우드** 🏆
- 이유: 쉽고, 저렴하고, 빠름
- 한국어 지원으로 문제 해결 용이

---

### 🎯 시나리오 2: 글로벌 서비스 / 기업용

**상황**:
- 여러 나라 사용자 대상
- AWS 서비스 활용 예정 (S3, Lambda, RDS 등)
- 장기적 확장성 중요
- 클라우드 경험 있음

**추천**: **AWS** 🏆
- 이유: 글로벌 인프라, 무한 확장성
- 다양한 서비스 통합 가능

---

### 🎯 시나리오 3: 교육용 / 테스트용

**상황**:
- 클라우드 학습 목적
- 짧은 기간만 사용
- 비용 최소화

**추천**: **네이버 클라우드** 🏆
- 이유: 크레딧 100,000원으로 1.5개월 무료
- AWS 프리티어는 DocuNova에 부적합 (메모리 부족)

---

### 🎯 시나리오 4: 이미 AWS 사용 중

**상황**:
- 다른 서비스를 이미 AWS에서 운영 중
- AWS 계정 및 VPC 설정 완료
- 통합 관리가 필요

**추천**: **AWS** 🏆
- 이유: 기존 인프라와 통합 관리
- 네트워크 통신 용이

**방법**: 동일
